# gravity

## A soundboard app written using electron.


### Install dependencies

```
$ npm install
```

### Run

```
$ npm start
```

### Build installer(s)

```
$ npm run build-linux
$ npm run build-osx
$ npm run build-win
```

Builds the app for macOS, Linux, and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
